TECH DIRECTORY Asiaとは

媒体概要

アジアに進出する製造業を後押しする「技術データベースサイト」

TECH DIRECTORY Asiaは、イプロスで培った成功ノウハウを海外向けに展開した、技術データベースサイトです。

アジア7ヵ国（タイ、ベトナム、マレーシア、シンガポール、インドネシア、フィリピン、中国）の各国サイトで現地語+日本語の完全二言語対応を行っているため、現地キーマンから問い合わせを獲得し、営業活動につなげることができます。

TECH DIRECTORY Asiaを使えば、現地での集客・販促活動を効率的に行うことができます。

国内向けイプロス同様に、会社情報、製品情報、カタログなどの必要事項を掲載できる上、現地語での検索エンジン対策が最適化された貴社のホームページの代替としてご利用いただけます。
また、既にホームページがある企業様には、誘導元としてご活用いただくことも可能です。

ぜひ海外での貴社の集客・販促活動にお役立てください。

販促支援会員のご案内

引き合い獲得に力を発揮する、各種機能・サービス

■利用費

販促支援会員に登録すると、導入実績・事例掲載、サイト内での優遇掲載、情報掲載ページ数増など、引き合い獲得の効率が高まる各種機能をご利用いただけます。
また、販促支援会員向けに、ページに掲載する際の翻訳サポートも格安でご提供させていただいておりますので、是非ご利用ください。

また、複数可国を同時に登録していただくと、最大30％月額利用費が割引になり、お得です。
